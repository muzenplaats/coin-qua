## Prompt:
金錢卦並加上天地之數，所以每卜一個卦只有一個之卦的演算法是甚麼?

## Completion:
金錢卦（也稱為搖錢卦）演算法的核心在於將「概率」轉化為「象數」。傳統的大衍之筮程序繁瑣，而金錢卦簡化了過程，同時透過納入「天地之數」的邏輯，可以精確地推導出變爻。

要達到「每卜一卦只有一個之卦」的效果，其演算法主要分為以下三個核心步驟：

### 1. 三錢成爻：單爻的機率分布

金錢卦使用三枚銅錢（通常以有字一面為「陰/背」，無字一面為「陽/面」）。根據《易經》天圓地方、參天兩地的原則，設定陽面為 **3**，陰面為 **2**。

每搖一次，將三枚銅錢的數值相加，會得到四種結果：

* **3 + 3 + 3 = 9**：老陽（重），為變爻。
* **2 + 2 + 2 = 6**：老陰（交），為變爻。
* **3 + 3 + 2 = 8**：少陰（拆），為靜爻。
* **3 + 2 + 2 = 7**：少陽（單），為靜爻。

透過連續搖六次，由下往上排列，即組成「本卦」。

---

### 2. 天地之數與動爻判定

在演算法中，「天地之數」通常指的是《繫辭傳》所言：「天數五，地數五，五位相得而各有合。天數二十有五，地數三十，凡天地之數五十有五。」

若要確保「只有一個之卦」，關鍵在於如何從六個爻位中判定「發動」的那一個。常見的邏輯有兩類：

* **自然發動法**：
在六次投擲中，如果只有一個爻出現 6 或 9，該爻即為「動爻」。此爻由陰變陽或由陽變陰，產生的新卦即為「之卦」。
* **加總除餘法（進階演算法）**：
若六次投擲中出現多個動爻或全無動爻，則會引入「天地之數」進行運算：
1. 將六次投擲所得的總數（例如：7+8+9+7+6+8 = 45）加總。
2. 將總和加上當時的「時數」（如子時為 1，丑時為 2）。
3. 將此數除以 6，取其**餘數**。
4. 餘數即代表第幾爻發動（若餘 0 則視為第六爻）。



---

### 3. 演算法邏輯總結

若要開發一個自動化程序來實現此邏輯，其演算法偽代碼如下：

1. **初始化**：設定天數（1, 3, 5, 7, 9）與地數（2, 4, 6, 8, 10）之和為 55。
2. **生成本卦**：
* 循環 6 次，每次隨機生成三個 {2, 3} 的數值。
* 計算單次總和 $S$。
* 若 $S=6$ 或 $9$，標記為潛在變爻。


3. **計算唯一動爻位**：
* 計算六爻總值 $Sum$。
* 動爻位 $N = (Sum + \text{額外參數}) \pmod 6$。
* （註：額外參數可選用天地之數 55 或當前時間）。


4. **生成之卦**：
* 僅將本卦中第 $N$ 爻的陰陽屬性反轉。
* 輸出：本卦、動爻、之卦。



這種演算法確保了即使在自然隨機下出現多個老陰老陽，最終也能透過「除餘法」鎖定一個核心動爻，從而指向唯一的「之卦」進行斷事。
